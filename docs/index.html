Abstract

MoNaVLA는 로봇의 동적 네비게이션을 위해 설계된 차세대 Vision-Language-Action 모델입니다. 시각적 편향(Visual Bias) 문제를 해결하기 위해 텍스트-로짓 민감도 분석을 수행하였으며, 가장 최신화된 Exp 09에서는 8-Class Discretization과 Counterfactual Bootstrapping 기법을 적용하여 제자리 회전(Turn-in-place)과 같은 미세한 기동을 극적으로 향상시켰습니다.

Methodology: Optimizing Action Policies (Exp 09)

Counterfactual Bootstrapping

제자리 회전(Class 6, 7)은 에피소드 극초반에만 짧게 발생하여 데이터 불균형을 야기합니다. 이를 해결하기 위해 데이터로더 수준에서 15% 확률로 "제자리 회전 명령 + 회전 액션"을 강제로 주입하여, 명령어와 액션 간의 Grounding 결합도를 물리적으로 강화합니다.


8-Class Discretization

기존의 좌/우 이동(Strafe)과 제자리 회전(Turn-in-place)을 분리하여 더욱 정밀한 3DOF 기동을 구현합니다.

Class ID Action Name Target Instruction (Example)
0STOP"Halt in front of the gray basket."
1Forward"Navigate straight forward."
2 / 3Strafe (L/R)"Move left to the obstacle."
4 / 5Diagonal"Angle toward left-front side."
6 / 7Turn (L/R)"Rotate toward the basket in place."
