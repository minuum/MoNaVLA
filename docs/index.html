핵심 전략 : Bootstrapping & 8-Class Discretization

  • 역사실적 주입(Counterfactual): 15%의 확률로 "회전" 명령과 액션을 강제 주입하여 데이터 불균형 극복
  • Grounding 강화: 한/영 병기 지시문을 통해 VLM의 명령 이해도 최적화
