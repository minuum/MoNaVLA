Current Public Reading

MoNaVLA는 모바일 로봇 내비게이션을 위한 vision-language-action 연구 트랙입니다. 현재 공개 결과 기준으로는, 거대한 end-to-end policy를 직접 학습시키는 것보다 grounding을 명시적으로 꺼내고 작은 action head로 연결하는 decomposition 접근이 더 안정적으로 작동합니다. 따라서 메인 baseline은 현재 Exp14 Step 2이며, 최신 gate 실험인 Exp18도 PM 27.62% / closed-loop 11.1%로 실패했습니다. 즉 현재 공개 기준에서는 decomposition 쪽 결론이 더 강해졌습니다.

Current Technical Interpretation

  • Decomposition currently beats checked end-to-end policy
    Exp14 Step 2: PM 75.9%, closed-loop 66.7% vs Exp11: PM 58.6%, closed-loop 0%
  • Image features are doing most of the useful work
    Feature ablation shows image-only is nearly as strong as bbox+image, while bbox-only is weaker and unstable.
  • Text attention remains the main failure mode
    Pure HF Kosmos keeps about 22.6% text attention, but Google-robot policy variants including head-only Exp15 still collapse to near-zero text attention.
  • Proxy branch is the only live challenger right now
    Exp19 reached 76.58% PM and 55.6% closed-loop, so it clearly beats recent end-to-end branches but still remains below Step2 at 66.7%.
