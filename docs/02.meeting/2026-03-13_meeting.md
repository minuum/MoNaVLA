# 주간 미팅 보고서 (2026-03-13)

## 1. 주요 보고 사항

### V4-EXP01 학습 완료 및 성능 검증
- **학습 개요**: V3-EXP08 가중치를 기반으로 신규 V3 데이터셋(`mobile_vla_dataset_v3`)을 활용한 Fine-tuning 수행.
- **주요 성과**:
    - **빠른 수렴**: 총 7 Epoch 학습 중 **Epoch 3에서 최적 성능(Val Loss: 2.816)** 달성 후 조기 종료(Early Stopping).
    - **학습 효율성**: 전체 학습에 약 1.5시간 소요 (평균 13분 30초/Epoch), LoRA 기반 최적화의 유효성 확인.
    - **데이터 전략**: V3 종합 데이터셋을 사용하되, 장애물이 없는 상황에서의 직진성(V4 Scenario)을 집중 학습하여 조향 진동 문제 해결 시도.
- **데이터 관리**: 최적 가중치(Best Checkpoint)를 추론 전용 서버(`soda@100.85.118.58`)의 `MoNaVLA/` 디렉토리로 전용 전송 중.

### 논문 작성 현황 (`2603 paper v3.md`)
- **아키텍처 기술**: 하이브리드(Kosmos-2 + CLIP) 구조 및 Policy-Head 기반의 연속 액션 제어 논리 명문화.
- **실험 결과 반영**: 기존 V3-EXP08의 PM(Perfect Match) 100% 달성 기록을 바탕으로, 신규 V4 실험의 정량적 수치(Success Rate) 업데이트 준비 중.
- **최적화 증거**: Jetson Orin NX(16GB) 환경에서의 메모리 점유(약 7.4GB) 및 실시간성(10Hz) 데이터 확보.

### 모델 정량 평가 결과 (V4-EXP01)
- **평가 지표**: Perfect Match (PM), Directional Match (DM)
- **대상**: `mobile_vla_dataset_v3` (Validation Split, 87 samples)
- **결과**:
    - **PM (Perfect Match)**: **100.00%** (클래스 완전 일치)
    - **DM (Directional Match)**: **100.00%** (조향 방향 일치)
- **분석**:
    - `Val Loss (2.816)`은 10스텝(Chunk)의 누적 손실값으로, 스텝당 평균 손실은 약 0.28 수준임.
    - 이는 100%에 가까운 분류 정확도와 정합하며, 모델이 V3 데이터셋의 주행 정책을 완벽하게 학습했음을 시사함.
    - 향후 일반화 성능 검증을 위해 Basket 데이터셋 등 외부 데이터 평가 필요.
