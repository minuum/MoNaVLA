## 2. Exp53 아키텍처

```
이전 (Exp49):
  Kosmos-2 CLIP[전체 frozen] → vision_feat(1024)
  + bbox_history(32) + goal_pos(3) → MLP → action

현재 (Exp53):
  Kosmos-2 CLIP[0-15 frozen | 16-23 LoRA r=16] → vision_feat(1024)
  + bbox_history(32) + goal_pos(3) → MLP → action
```

**LoRA 스펙:**
- 대상 레이어: `vision_model.model.encoder.layers[16:24]`
- Target modules: `q_proj`, `v_proj`
- r=16, alpha=32, dropout=0.05
- 학습 파라미터: **524,288개** (전체 304M 중 0.17%)
- 추론 오버헤드: merge 시 0% (동일 속도)

**왜 16~23인가:**
- CLIP ViT 24개 블락 중 마지막 8개 = semantic alignment layer
- 현재 Kosmos-2가 gray basket을 "gray metal box", "trash can"으로 오인식
- 이 레이어에 LoRA → 우리 환경 바스켓을 명시적으로 학습

## 3. 핵심 검증 질문

> **"LEFT를 외운 건가, 박스를 본 건가?"**

9 에피소드(좌/중/우 × 3)로 3번 성공했지만, 아래 중 어느 이유인지 불명확:
- (A) `"basket under left"` 텍스트 패턴 암기 → left 액션
- (B) CLIP이 박스 위치를 시각적으로 인식 → 방향 결정

**검증 방법:**
| 프롬프트 | 방향 | 시도 | 성공 |
|----------|------|------|------|
| `"basket under left"` (방향어 포함) | 왼쪽 | 5회 | ? |
| `"basket in center"` | 중앙 | 5회 | ? |
| `"basket under right"` | 오른쪽 | 5회 | ? |
| **`"go to the box"`** (방향어 없음) | 왼쪽 | 5회 | ? |
| **`"go to the box"`** | 중앙 | 5회 | ? |
| **`"go to the box"`** | 오른쪽 | 5회 | ? |

방향어 없이도 성공 → **박스 시각 인식 근거**  
방향어 있을 때만 성공 → **텍스트 패턴 암기 의심**
