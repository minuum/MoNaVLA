## 포텐셜 평가 (Exp17)

### 긍정 신호
✅ Exp16 collapse의 원인 명확: FORWARD bias (74.4%)  
✅ 명시적 균등 샘플링으로 bias 제거 시도  
✅ 학습 초기 수렴 양호 (3.715 → 2.270)

### 부정 신호
❌ Backbone 미해결: Google-robot text attention = 0% (고정)  
❌ End-to-end 구조 한계: Exp11 증명 (val_loss 좋음 → PM 58.6% → closed-loop 0%)  
❌ val_loss ≠ 실제 성능 (과거 사례 다수)

### 예측
- **낙관:** PM이 60~70% 수준으로 개선될 가능성
- **현실적:** Decomposition (Exp14 Step2: 75.9% PM, 66.7% closed-loop)을 이기기 어려움
- **교수님 프로토콜:** Step 3 검증으로서의 의미 있음 (data bias vs architecture bias 구분)
